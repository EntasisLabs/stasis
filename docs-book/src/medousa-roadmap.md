# Medousa Roadmap

## Product Identity

Medousa is a continuous web researcher agent built on Stasis runtime orchestration.

## Why This Exists

- Real product direction first.
- Demo value emerges from real operator workflows.
- Reliability, lineage, and policy are first-class.

## Milestones

- M0: Product skeleton and docs alignment.
- M1: Ask and report vertical slice.
- M2: Recurring research topics.
- M3: Guarded generated workflow authoring.
- M4: Operator CLI and run inspection UX.

## Next Slice

Build medousa daemon and medousa cli entrypoints, then deliver one end-to-end ask flow with report and lineage output.
